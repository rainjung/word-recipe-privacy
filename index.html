개인정보 처리 방침

영단어 레시피 (Word Recipe)  ·  시행일: 2026년 5월 22일

영단어 레시피(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 및 관련 법령에 따라 적법하게 처리·보호합니다. 본 방침은 앱이 어떤 정보를 수집하고, 어떻게 이용하며, 어떻게 보호하는지 설명합니다.

1. 수집하는 개인정보 항목

앱은 다음 두 가지 항목만 외부 서버(Supabase)에 저장합니다.

① 익명 사용자 식별자 앱 최초 실행 시 자동 발급되는 임의의 UUID. 이름·이메일·전화번호·생년월일 등 신원 정보는 일절 수집하지 않습니다.
② 결제 정보 Apple In-App Purchase 거래 식별자(transactionId), 상품 ID, 영수증 데이터. 신용카드 번호 등 실제 결제 수단은 Apple 이 직접 처리하며 앱 서버에 저장되지 않습니다.
수집 방법 앱 실행 시 자동 발급(익명 인증), 결제 시 자동 생성
학습 데이터(학습한 단어·진행 현황·통계)는 사용자 기기 내부 SQLite 데이터베이스에만 저장되며, 외부 서버로 전송되거나 제3자에게 제공되지 않습니다.

2. 개인정보 이용 목적

  • 익명 사용자 식별자: 결제 권한(평생 이용권 등) 매칭 및 환불 시 권한 해제
  • 결제 영수증: Apple App Store / Google Play 영수증 서버 검증 및 환불 처리
  • 서비스 오류 처리 및 고객 지원
  • 법적 의무 이행 및 분쟁 해결

3. 제3자 제공 및 수탁 처리

제공·수탁 업체 제공 목적 제공 항목
Supabase, Inc. (미국) 데이터베이스 호스팅, 영수증 서버 검증, 환불 처리 익명 사용자 식별자, 결제 영수증
Apple Inc. (미국) App Store 인앱 결제 및 영수증 검증, 환불 알림(ASSN V2) 결제 영수증
Google LLC (미국) Google Play 인앱 결제 및 영수증 검증, 환불 알림(RTDN) 결제 영수증

위 업체 외 제3자에게 개인정보를 제공·판매하지 않으며, 광고망·분석 도구에 사용자 데이터를 공유하지 않습니다.

4. 보관 기간 및 파기

구분 보관 기간
익명 사용자 식별자 사용자 요청 시 즉시 파기 (법령에 따른 의무 보관 기간 예외)
결제 관련 기록 「전자상거래 등에서의 소비자 보호에 관한 법률」에 따라 5년 보관 후 파기
학습 데이터 (기기 내) 앱 삭제 또는 사용자가 앱 내 '학습 진행상황 초기화' 실행 시 즉시 파기

파기 방법: 전자 파일은 복구 불가능한 기술적 방법으로 영구 삭제합니다.

5. 사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 열람: 본인의 개인정보 처리 현황 확인
  • 정정·삭제: 부정확한 정보의 수정 또는 익명 사용자 식별자·결제 기록 삭제 요청
  • 처리 정지: 특정 목적의 개인정보 처리 중단 요청
  • 이동 요청: 본인 데이터의 기계 가독 형식 제공 요청
  • 데이터 삭제: 앱 삭제 시 기기 내 학습 데이터는 즉시 삭제됩니다. 외부 서버 저장 정보는 아래 문의처로 요청 시 처리됩니다.
권리 행사는 아래 문의처로 이메일을 보내시면 10영업일 이내 처리됩니다.

6. 아동 개인정보 보호

앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만임을 확인한 경우 즉시 해당 정보를 삭제합니다.

7. 개인정보 보호 조치

  • 저장 데이터 암호화 (Supabase 인프라 수준 AES-256)
  • 전송 구간 HTTPS/TLS 적용 (HTTP 평문 통신 차단)
  • 인증 토큰은 기기 내 보안 저장소(iOS Keychain / Android Keystore)에만 보관
  • 접근 권한 최소화 (Row-Level Security 정책 적용 — service_role 키는 클라이언트에 노출되지 않음)
  • 광고 추적 SDK 및 외부 분석 도구를 일절 사용하지 않습니다.
